# 工具源码说明

本文件夹包含用于生成和导出测试数据的两个 TypeScript 脚本。

## seed.ts — 数据生成主脚本

**功能：** 自动生成 100,000+ 条具有完整家族层级结构的测试数据

**运行方式：**
```bash
# 在项目根目录执行
npm run db:seed
# 等价于：npx tsx scripts/seed.ts
```

**生成内容：**
- 3 个测试用户账号（密码 bcrypt 哈希存储）
- 10 个族谱（包含一个 50,000+ 成员的大型族谱）
- 每个族谱至少 30 代传承
- 亲子关系（parent_child 表）
- 婚姻关系（marriages 表）
- 同步生成 `scripts/members_export.csv` 文件（用于 COPY 导入演示）

**核心算法：层级扩展法**
1. 创建第1代始祖夫妇
2. 每代人从上一代中繁衍，每对父母生 2-5 个孩子
3. 批量插入（每批 500 条，避免一次性插入太多）
4. 为男性成员创建配偶（族外婚，随机姓氏的女性成员）
5. 重复直到总人数达到目标值且代数满足最低要求

## export-branch.ts — CSV 导出脚本

**功能：** 将指定成员的所有后代递归导出为 CSV 文件，演示 PostgreSQL COPY 命令的等效操作

**运行方式：**
```bash
# 在项目根目录执行（替换为真实成员ID）
npx tsx scripts/export-branch.ts clx1abc...
```

**输出：**
- `scripts/branch_前8位ID.csv`：包含该成员及其所有后代的信息
- 终端打印等效的 PostgreSQL `\COPY` 命令（可直接复制到 psql 执行）

**核心 SQL（递归 CTE）：**
```sql
WITH RECURSIVE descendants AS (
  SELECT m.* FROM members m WHERE m.id = '目标ID'
  UNION ALL
  SELECT m.* FROM members m
  JOIN parent_child pc ON pc.child_id = m.id
  JOIN descendants d   ON pc.parent_id = d.id
)
SELECT * FROM descendants ORDER BY depth, generation, name
```
